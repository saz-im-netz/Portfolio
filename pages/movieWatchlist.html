My Movie Watchlist

Projekt Übersicht

Das Projekt "My Movie Watchlist" besteht aus zwei Seiten: die Filmsuche und die Filmliste. Über den Filmtitel können Filme aus der "Open Movie Database" abgefragt werden, die auf der Filmsuche angezeigt werden. Diese können anschließend und auf der Filmliste gespeichert und jederzeit aufgerufen werden.

Verwendete Sprachen und Resourcen

HTML
CSS
JavaScript
OMDb API
LocalStorage
Github
Netlify
